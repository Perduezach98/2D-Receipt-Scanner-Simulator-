# Receipt Scanner Simulator (Godot 4)

A lightweight Godot 4 desktop app that simulates receipt uploads and OCR extraction.

## Features
- Upload receipt images (png, jpg)
- Simulated OCR progress
- Extracted fields: merchant, date, total, category
- Receipt history list + detail viewer
- Export all receipts to JSON

## How to Run
1. Install Godot 4.x
2. Open the project folder in Godot
3. Run `scenes/Main.tscn`

## Notes
This project simulates OCR and does not use an external OCR engine.
